# Rakan Alhakim — Portfolio

A world-class personal portfolio website featuring a luxurious glass design system, smooth animations, and premium interactions.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React** (Icons)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Features

- **Glass Design System**: Frosted glass cards with blur effects and elegant borders
- **Smooth Animations**: Framer Motion powered interactions and transitions
- **Responsive Design**: Optimized for all screen sizes
- **SEO Optimized**: Complete metadata and semantic HTML
- **Performance**: Fast loading and optimized builds

## Design Philosophy

The portfolio follows a **glass luxury minimalism** aesthetic:

- Frosted glass cards with backdrop blur
- Subtle transparency layers
- Animated gradient backgrounds
- Thin elegant borders
- Smooth depth shadows
- Premium spacing and typography

Everything is designed to feel intentional, premium, and alive.
